# Provide Insights to Management in Consumer Goods Domain
#  PROBLEM 
Atliq Hardware is a leading manufacturer of computer hardware in India and other nations, but it struggles to make prompt and wise choices based on data. The management intends to add more junior data analysts to the data analytics team. Tony Sharma, the head of data analytics, conducts a SQL challenge to identify applicants with both technical and soft skills.

#  TOOLS & APPROACH 
MySQL, PowerBI, Canva, Adobe Express, OBS, and Droidcam.

路 MySQL - Concepts like Joins, CTEs, Subqueries, Window functions, String functions, Numeric functions, Date functions, etc. 路 PowerBI - Dax and visuals like Sunburst Chart, Bar Chart, Matrix, Line Chart, Maps, Donut Chart, etc.

#  FINDINGS & SOLUTIONS 
路 Findings - Important metrics such as Gross Profit, Contribution %, Order by Overall Amount Sold, Average Gross Sales, Average Manufacturing Cost, etc. 路 Solutions - Increasing production rate by enticing customers with memberships, gift certificates, bundled deals, student discounts, and other special offers.
